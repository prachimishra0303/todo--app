To-Do List Web App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Completed)
- Data stored using localStorage

Technologies Used

- HTML
- CSS
- JavaScript

How to Run

1. Open the project folder
2. Open "index.html"
3. Run in browser or preview mode

Project Purpose

This project was created as part of a web development internship task to demonstrate CRUD operations, DOM manipulation, filtering, and localStorage usage.
